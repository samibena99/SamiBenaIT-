# 🖥️ IT Asset Tracker

## Overview
A structured spreadsheet-based system designed to manage, track, and audit all IT assets across departments — ensuring accurate inventory and reducing asset loss.

## Problem
In organizations without a formal tracking system, IT assets (laptops, desktops, printers, peripherals) can go missing or become unaccounted for — leading to wasted budget and compliance issues.

## Solution
Developed a centralized Excel/Google Sheets tracker integrated with Microsoft 365 to log all assets, their assigned users, status, and maintenance history.

## Features
- 📋 Asset registration (device type, serial number, brand, model)
- 👤 User assignment and department mapping
- 🔁 Checkout/check-in history log
- 🛠️ Maintenance and repair tracking
- 📊 Summary dashboard with asset status overview
- 📅 Warranty expiry alerts

## Technologies Used
- Microsoft Excel / Google Sheets
- Microsoft 365 (SharePoint for shared access)
- Power Query for data aggregation

## Outcome
- Reduced untracked assets by ~80% in the first quarter of deployment
- Enabled faster hardware audits during compliance reviews
- Saved approximately 3–4 hours/week in manual inventory checks

## Sample Fields

| Field | Description |
|-------|-------------|
| Asset ID | Unique identifier per device |
| Category | Laptop / Desktop / Printer / Peripheral |
| Brand & Model | e.g., Dell Latitude 5520 |
| Serial Number | Manufacturer serial number |
| Assigned To | Employee name and department |
| Status | Active / In Repair / Retired / Available |
| Location | Office / Remote / Storage |
| Purchase Date | Date of procurement |
| Warranty Expiry | End of warranty date |
| Last Audited | Date of last physical check |
