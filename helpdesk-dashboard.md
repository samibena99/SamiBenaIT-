# 📊 Helpdesk KPI Dashboard

## Overview
A reporting dashboard built to track IT helpdesk performance metrics, ticket SLA compliance, and resolution trends — enabling data-driven decisions for IT management.

## Problem
IT managers lacked visibility into ticket backlogs, resolution times, and recurring issues, making it difficult to allocate resources efficiently or identify problem areas.

## Solution
Built a Power BI / Excel dashboard that pulls data from the ticketing system and CRM to visualize key IT support metrics in real time.

## Features
- 🎫 Open vs Resolved ticket count (daily/weekly/monthly)
- ⏱️ Average response time and resolution time per category
- 🔴 SLA breach alerts and aging ticket tracker
- 🏷️ Issue categorization breakdown (hardware / software / network / access)
- 👤 Technician workload distribution
- 📈 Trend analysis for recurring issues

## Technologies Used
- Microsoft Excel (Pivot Tables, Charts)
- Power BI (Data visualization)
- Microsoft 365 (SharePoint, Teams integration)
- CRM data exports

## Outcome
- Reduced average ticket response time by **20%** through better workload visibility
- Improved lead/customer data conversion by **8–12%** via CRM reporting
- Supported management reporting with ready-to-share weekly snapshots

## Key Metrics Tracked

| KPI | Target |
|-----|--------|
| First Response Time | < 1 hour |
| Ticket Resolution Time | < 4 hours (P1), < 24 hours (P2) |
| SLA Compliance Rate | ≥ 95% |
| Ticket Backlog | < 10 open tickets |
| Customer Satisfaction | ≥ 4.5 / 5.0 |
